JavaScript - d3-dchart
======================

A simple JavaScript D3.js chart. 



Install
-------


### Script

```html
<script src="/path/to/d3-chart.js"></script>
```


### Browserify

    npm install --save d3-chart


### Component

    component install jprichardson/d3-chart
